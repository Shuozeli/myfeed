# Tech Radar

Read the feed data from `myfeed dump --hours 24 --site hackernews --site reddit --site zhihu`
and produce a technology-focused summary.

## Output Format

**New/Notable** (technologies, tools, papers, launches mentioned for
the first time or gaining traction)

**Discussions** (debates, opinions, experience reports about existing
technologies)

**Career** (job market signals, hiring trends, interview experiences)

## Rules

- Focus on technical content. Skip politics, lifestyle, memes.
- For papers/launches, include the URL.
- For Zhihu items, translate the title and give a 1-sentence summary.
- Note if something is generating controversy (high comment count
  relative to upvotes).
